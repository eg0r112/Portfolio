# Портфолио

Открой `index.html` в браузере или запусти локально:

```bash
cd portfolio
npx serve .
```

## Публикация на GitHub Pages

1. Создай репозиторий `eg0r112.github.io` (или любой другой)
2. Скопируй содержимое папки `portfolio` в корень репозитория
3. Settings → Pages → Source: main branch
4. Сайт будет на `https://eg0r112.github.io`
