01 Обо мне

Занимаюсь backend-разработкой уже 2 года. Собираю сервисы целиком: REST API, PostgreSQL, фоновые воркеры, очереди, кэш, JWT/RBAC, интеграции с внешними системами, Docker и CI.

Флагманский pet-проект — IRP (Incident Response Platform): алерты → инциденты → outbox → RabbitMQ → уведомления в Telegram и Jira, on-call бот, admin-панель, OpenTelemetry.

Pet-проект Team Availability Scheduler: команды → отпуска и отсутствия → расчёт «кто доступен» → on-call смены → outbox → Telegram-бот (/status, /working, /away) и Jira-тикет при дежурстве, Web UI, integration tests.

Некоммерческий опыт для Fun&Sun — backend и Telegram Mini App для поиска туров. Пишу unit- и integration-тесты, работаю со Swagger и GitHub Actions.
